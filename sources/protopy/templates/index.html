¿Que es Protopy?

Protopy es una biblioteca Open Source en JavaScript para el desarrollo de aplicaciones web dinámicas.

Basada en Python, aporta un enfoque modular para la inclusión de código, orientación a objetos, manejo de AJAX, DOM y eventos.
